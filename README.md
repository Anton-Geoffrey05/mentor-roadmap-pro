# Mentor Roadmap Pro

AI-powered career roadmap generator for freshers. Answer a 2-minute assessment and get a
personalized, week-by-week learning plan — skill-gap analysis, portfolio projects,
certifications, and curated free resources — generated by Gemini and tailored to your
degree, current skills, and target role.

## Features

- **Career assessment** — degree, branch, skills, target career, study hours, learning
  style (React Hook Form + Zod validation)
- **AI roadmap generation** — Gemini builds a structured weekly plan with milestones,
  projects (with tech stacks), certifications, and free resources
- **Skill gap analysis** — readiness score and missing skills ranked by learning order
- **Smart caching** — identical assessments are served from Postgres instead of
  re-calling the LLM (SHA-256 cache key per user + input)
- **Authentication** — Clerk (sign up / sign in / protected routes)
- **Light & dark theme**, responsive dashboard, toast notifications, loading skeletons

## Architecture

```
React 19 + Vite + TS ──► Supabase Edge Function (Deno) ──► Gemini API
        │                        │
      Clerk JWT ◄── verified via JWKS (jose)
                                 │
                          Postgres (RLS) — profiles, career_goals,
                          roadmaps, milestones, projects
```

Security-first design decisions:

- The Gemini API key lives only in edge-function secrets — it never reaches the browser
- Clerk JWTs are verified server-side against Clerk's JWKS before any DB access
- All reads/writes go through the edge function with response-shape validation;
  the client never queries tables directly
- Row-level security enabled on every table

## Tech stack

React 19 · TypeScript (strict) · Vite · Tailwind CSS v4 · Framer Motion · TanStack
Query · React Hook Form + Zod · Clerk · Supabase (Postgres, Edge Functions) · Gemini

## Getting started

```bash
npm install
cp .env.example .env   # fill in your Clerk + Supabase keys
npm run dev
```

Backend setup (once):

```bash
supabase link --project-ref <your-project-ref>
supabase db push
supabase functions deploy generate-roadmap --no-verify-jwt
supabase secrets set GEMINI_API_KEY=<key> SUPABASE_SERVICE_ROLE_KEY=<key> CLERK_JWKS_URL=<url>
```
